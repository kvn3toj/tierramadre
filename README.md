# Tierra Madre Studio

Internal Advertising Agency Tool for Colombian Emeralds.

**"Esencia y Poder"** - The essence and power of Colombian emeralds.

## Features

- **AI-Powered Name Generator**: Smart suggestions from 80+ curated Tierra Madre names
- **Batch Upload**: Upload multiple emerald photos at once
- **Individual Upload**: Detailed metadata for single emeralds
- **PDF Catalog Export**: Generate professional catalogs
- **Instagram Planner**: Visual 3x3 grid preview and content calendar

## Tech Stack

- React 18 + TypeScript
- Vite
- Material-UI v6
- LocalStorage persistence
- No external API required (smart local generator)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
